About This Tool

The EPS Citation Generator builds standardized curriculum-alignment citations for proposals, RFP evidence tables, and compliance documentation. Assemble each citation field by field, save it to your library, then tag, search, copy, or export to CSV.

  1. Select an EPS Program. The Unit / Lesson / Task fields automatically relabel to match that program's structure.
  2. Choose a Material Type. Toggle Abbreviate to use the short key (e.g., TE) instead of the full name.
  3. Add the details — standards code, page number(s), format, delivery method, audience, evidence type, and location notes.
  4. Review the Live Preview, add optional tags, and Save Citation.
  5. Manage your library on the Builder tab — search, filter, tag, copy, or export to CSV.
Citation Structure

[Standards Code], [EPS Program Name], [Material Type], [Unit], [Lesson], [Task], [Page Number(s)], [Format], [Audience], [Evidence Type], [Location Notes & Cross-References]. See: ([Evidence Web Link (URL)])

  1. Standards Code: The Educational Standards Code this citation is related to. Field is Recommended but Optional for organization when generating several citations.
  2. EPS Program Name: This field is a Quick-Fill (see Program Hierarchy list below) and a manual program naming field. The Unit / Lesson / Task fields automatically relabel to match that program's structure if a quick fill is selected.
  3. Material Type: See the Material Types (32 total types) in the table towards the bottom of this page. Check the Abbreviate box to use the abbreviated material name (e.g., TE instead of Teacher Guide/Edition) instead of the full material name.
Program Structure Hierarchy

To remove ambiguity between program structures, a standardized format was used to ensure everything is cleanly cited:
Unit → Lesson → Task

  1. Unit: Field automatically relabels to match the EPS Program Name structure. Level / Skill / Skill Group / Book / Set
  2. Lesson: Field automatically relabels to match the EPS Program Name structure. Lesson / Word List / Book / Reader
  3. Task: Field automatically relabels to match the EPS Program Name structure. Step / Activity / Exercise / Read / Reading Task
  1. Page Number(s): Allows for single page and multi-page input. Will add p. for single page, and pp. for multipage citations automatically.
  2. Format: If the alignment requires the material format the allows for Print, Digital, or Hybrid to be added. It is blank by default.
  3. Audience: Optional field, where the choices are Teacher or Student. This is related to the Evidence Type field. See table below for detailed explanation of Field Definitions.
  4. Evidence Type: Optional field, where the choices are Narrative or Activity. This is related to the Audience field. See table below for detailed explanation of Field Definitions.
  5. Location Notes & Cross-References: Short directions to locate cited evidence, cross-reference to other artifact type and/or a very brief narrative if necessary.
  6. Evidence Web Link (URL): Must provide a direct link to evidence of the citations alignment to the Educational Standards Code being cited.
Field Definitions
Audience
Teacher

Artifacts designed for the educator to facilitate or guide instruction — e.g., explicit scripts, instructional rationales, modeling guides, lesson overviews, and progress-monitoring forms.

Student

Artifacts designed for student interaction where the student is the primary actor — e.g., workbooks, student text, digital activities, and worksheets.

Evidence Type
Narrative Input

Represents the "Input" phase of learning — the opportunity for the teacher to teach the concept, or for the student to initially learn or read about it.

Activity Output / Application

Represents the "Output" / "Application" phase — the student actively practices, demonstrates, or applies the skill.

Program Hierarchy
